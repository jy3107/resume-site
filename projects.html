Projects

Iconic Reef Guardians - Data Automation and Processing

Using Excel VBA, I developed a script that would delete skewed/tester data and unnecessary columns when gathering impact metrics. The script would further reformat the raw data gathered from dive operators taking the training course. Then, the organized data can then be used to quickly design visual graphs to showcase the participants' improved understanding of coral restoration efforts from before to after taking the course.

Goal: Clean Seas Florida Keys - Data Analysis and Visualization

With reporting marine debris removal trips' information (pounds of debris removed, types of debris removed, etc.) to a data collection program called PADI AWARE, I was tasked to organize >10k unprocessed data year-to-date. With the cleaned data, I use data visualization to showcase the impact the program and dive operators have made so far in underwater debris removal through visual reports. Furthermore, I reformatted the data for a seamless transfer of data from PADI to a new data collection program called the Marine Debris Tracker App that allows for the data to be publicly accessible.
